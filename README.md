# AnagramGenerator

To use the script:
1. Run `anagram.py`
2. Follow the prompts in the terminal to enter letters to search for.
3. Type `exit` or `reset` on any of the prompts for letters to exit or reset the program.

The base dictionary can be easily swapped out by changing the file in `open_src_dict()`. Currently the program is using enable1.txt (from https://github.com/dolph/dictionary) as its dictionary.
